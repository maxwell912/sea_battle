Sea Battle

Rules: https://en.wikipedia.org/wiki/Sea_Battle

Key commands:
    KeyUp/Down/Right/Left - change cursor position
	Enter - shoot
	S - save game (file will appear in a current directory)
	Esc - exit game

Game launch:
	Run sea_battle.py with python
	
	Console arguments:
		-w - width of the game board (default=10)
		-hg - height of the game board (default=10)
		-s - file with saved game progress
